# Sloth

使用方式：

启动run.bat，会自动弹出一个下载视频的web界面，把各种视频链接放进去就行了。

![image-20250224171444823](./README.assets/image-20250224171444823.png)

# 使用修改

如果你自己使用，需要修改的内容：

`constant.py`

```
class Constant:
    DIR = "F:\Video"
    PROXY = 'http://127.0.0.1:4343'
```

改为你自己的`默认目录`和`代理（为了科学上网）`
